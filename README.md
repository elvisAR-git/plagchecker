## Libraries and technologies used

    1.  Mongo DB
    2.  Express.js
    3.  materialize.js

## Features

- sleek material UI
- Upload multiple files
- SPA feel
- file comparision
- similarity listing
- percentage unique
- database persistence
- similarities highlighting (styling)

---

## How to run

requirements::

node.js version 16.0.0

No database setup required!
run:: "npm install"
then:: "npm start"
